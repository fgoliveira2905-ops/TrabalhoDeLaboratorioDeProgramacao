Detailed Description

Declaração das funções auxiliares para manipulação de vetores e matrizes.

Este arquivo contém as funções principais usadas pelo programa principal para operações com vetores de inteiros, como cálculo de média, ordenação, filtragem e construção de matrizes.

mmc()

Calcula e imprime o mínimo múltiplo comum (MMC) de elementos consecutivos de um vetor.

Esta função recebe um vetor A de inteiros e calcula o MMC de cada par de elementos consecutivos, armazenando os resultados num vetor auxiliar B. Por fim, imprime todos os valores calculados.

Parameters
A- Vetor de inteiros fornecido pelo usuário.
tamanho- Número de elementos do vetor A.
Note
O vetor resultante B terá tamanho igual a tamanho - 1, pois cada elemento representa o MMC de um par de elementos consecutivos do vetor original.
Essa função depende de da função mmc() e mdc().

Soma o vetor original com o dobro de um segundo vetor fornecido pelo utilizador.

Esta função lê um novo vetor B com N elementos fornecidos pelo utilizador, duplica todos os seus valores e, em seguida, soma cada posição do vetor original A com o vetor B já transformado.

O resultado dessa soma é armazenado num vetor auxiliar e impresso no ecrã.

Parameters
AVetor original.
NNúmero de elementos dos vetores.
Note
O vetor inserido pelo utilizador é modificado para conter o dobro dos seus valores antes da operação de soma.
