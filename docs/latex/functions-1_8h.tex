\doxysubsection{Detailed Description}
Declaração das funções auxiliares para manipulação de vetores e matrizes. 

Este arquivo contém as funções principais usadas pelo programa principal para operações com vetores de inteiros, como cálculo de média, ordenação, filtragem e construção de matrizes.

Calcula e imprime o mínimo múltiplo comum (MMC) de elementos consecutivos de um vetor.

Esta função recebe um vetor {\ttfamily A} de inteiros e calcula o MMC de cada par de elementos consecutivos, armazenando os resultados num vetor auxiliar {\ttfamily B}. Por fim, imprime todos os valores calculados.


\begin{DoxyParams}{Parameters}
{\em A} & -\/ Vetor de inteiros fornecido pelo usuário. \\
\hline
{\em tamanho} & -\/ Número de elementos do vetor {\ttfamily A}.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
O vetor resultante {\ttfamily B} terá tamanho igual a {\ttfamily tamanho} -\/ 1, pois cada elemento representa o MMC de um par de elementos consecutivos do vetor original.

Essa função depende de da função \doxylink{functions-1_8h_aa3d7c552c82bafc4972a6cb1605b028e}{mmc()} e \doxylink{functions-1_8h_a0a30302f4720a1282e1b5fd5dbed015a}{mdc()}. 
\end{DoxyNote}
