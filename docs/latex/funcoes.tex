\chapter{Funções }
\hypertarget{funcoes}{}\label{funcoes}\index{Funções@{Funções}}
Esta página descreve as principais funções disponibilizadas pelo programa, todas elas declaradas em {\ttfamily \doxylink{functions-team-2_8h}{functions-\/team-\/2.\+h}} e implementadas no respetivo ficheiro {\ttfamily .c}. As funções operam sobre um array de inteiros e permitem ao utilizador executar operações de processamento, análise e construção de matrizes.

Lista de funções\+:


\begin{DoxyItemize}
\item {\ttfamily \doxylink{functions-team-2_8c_ac17adb8b1b8b675aa4566658becdaed9}{media\+Array()}} — calcula a média aritmética dos elementos do array.
\item {\ttfamily \doxylink{functions-team-2_8c_aaf4950dc84a65aaae3c7c46a55eb24e6}{maiores\+Valores\+Divisiveis\+Por5()}} — imprime valores maiores que 2 e divisíveis por 5.
\item {\ttfamily \doxylink{functions-team-2_8c_a886912dd80dd71c90cfb118a7304d34b}{vetor\+Decrescente()}} — ordena o array por ordem decrescente e imprime-\/o.
\item {\ttfamily \doxylink{functions-team-2_8c_ad9b24c08424a9ad689669a157de42df1}{matriz\+Dois\+Por\+Catorze()}} — constrói uma matriz 2×N usando o array original e o ordenado.
\item {\ttfamily \doxylink{functions-team-2_8h_a306e81b7dc4188518aed66d290bf2c46}{calculo\+Da\+Raiz\+Quadrada()}} — calcula e imprime a raiz quadrada para cada elemento.
\item {\ttfamily \doxylink{functions-team-2_8c_a2befbe8b2f5e26a7c293869fec2b1e87}{minimo()}} — devolve o menor valor presente no array.
\item {\ttfamily \doxylink{functions-team-2_8c_ab3f194b11c0a186933381df283a42c7d}{mmc\+Do\+Vetor()}} — calcula o MMC para cada par consecutivo do array.
\item {\ttfamily \doxylink{functions-team-2_8c_aa3d7c552c82bafc4972a6cb1605b028e}{mmc()}} — calcula o mínimo múltiplo comum entre dois inteiros.
\item {\ttfamily \doxylink{functions-team-2_8c_a0a30302f4720a1282e1b5fd5dbed015a}{mdc()}} — calcula o máximo divisor comum entre dois inteiros.
\item {\ttfamily \doxylink{functions-team-2_8c_a233cbe48916fe7cd74f20346b9cd152e}{soma\+De\+Vetores()}} — soma o array original com o dobro de um novo array introduzido.
\item {\ttfamily \doxylink{functions-team-2_8c_a614426b198a46a2adbdbd8633b2d3170}{Multiplicar\+Matrizes()}} — efetua a multiplicação entre duas matrizes.
\item {\ttfamily \doxylink{functions-team-2_8c_a3bb3ab256d91f1d2593f26e8502dfca8}{Construir\+Matriz()}} — constrói a matriz 14×14 resultante do produto externo entre dois arrays. 
\end{DoxyItemize}
