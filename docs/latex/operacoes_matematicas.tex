\chapter{Operações Matemáticas }
\hypertarget{operacoes_matematicas}{}\label{operacoes_matematicas}\index{Operações Matemáticas@{Operações Matemáticas}}
Esta página descreve todas as operações matemáticas realizadas no âmbito do programa, permitindo ao utilizador analisar o comportamento numérico do vetor.

Operações abordadas\+:


\begin{DoxyItemize}
\item {\bfseries{Cálculo da média\+:}} soma dos elementos dividida pelo número total.
\item {\bfseries{Cálculo da raiz quadrada\+:}} transformação de cada elemento para sua raiz.
\item {\bfseries{Mínimo dos elementos\+:}} identificação do menor valor presente.
\item {\bfseries{Ordenação\+:}} organização do vetor de forma crescente ou decrescente.
\item {\bfseries{MMC de pares de números\+:}} cálculo do mínimo múltiplo comum entre elementos consecutivos.
\item {\bfseries{Produto de vetores para formar matriz\+:}} criação de uma matriz 14x14 a partir do vetor inicial. 
\end{DoxyItemize}
