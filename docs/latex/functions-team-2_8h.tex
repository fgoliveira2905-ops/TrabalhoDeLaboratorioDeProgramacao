\doxysubsection{Descrição detalhada}
Declarações de funções auxiliares para manipulação de arrays e matrizes. 

Este ficheiro contém as interfaces das funções utilizadas no programa principal para leitura, processamento e apresentação de dados armazenados em arrays e matrizes de inteiros.

As funções incluem cálculos numéricos, operações sobre arrays, construção de matrizes e outras utilidades.
